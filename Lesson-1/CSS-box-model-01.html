The CSS box model is essentially a box that wraps around every HTML element. It consists of: content, padding, borders & margins.

Some HTML elements like the paragraph element, headers h1-h6 elements have margin wrapping around them by default. You must remember this when working with these elements. There will be scenarios where a developer may need to remove the default margin or maybe even increase it to taste.
